Face Recognition Brain

This Face Recognition React app utilizes the following technologies to produce a bounding box around a face given an image containing a face:

  • Clarifai API for facial recognition
  • Node.js and Express.js for backend server
  • PostgreSQL database to keep track of registered users
  • TS Particles API for background animation
  • Bcrypt API for password hashing
  • Knex API to connect to database

This was the final project for the Zero to Mastery web development course.
